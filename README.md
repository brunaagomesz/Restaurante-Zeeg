# ☕ ANTONIETT

Projeto de um site de vendas de canecas desenvolvido com foco em experiência do usuário, design moderno e funcionalidade.

---

## 📌 Sobre o projeto

A **ANTONIETT** é um projeto acadêmico desenvolvido durante a faculdade, com o propósito de atender às necessidades de um cliente real.

A proposta vai além da teoria, permitindo a aplicação prática dos conhecimentos adquiridos em sala de aula, além de promover a expansão das habilidades em desenvolvimento front-end.

O projeto também envolve colaboração em equipe, fortalecendo competências como comunicação, organização e trabalho conjunto.

---

## 🎯 Objetivos

* Criar uma loja virtual funcional para venda de canecas
* Atender requisitos reais de um cliente
* Aprimorar habilidades em desenvolvimento front-end
* Trabalhar em equipe de forma colaborativa

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 💻 Funcionalidades

* Exibição de produtos
* Layout responsivo
* Interface amigável e intuitiva
* Interações com JavaScript
